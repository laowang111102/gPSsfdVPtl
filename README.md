# SpringBoot+Vue酒店管理系统 java947

## 项目概述

酒店管理系统是基于SpringBoot和Vue开发的，包括前台门户和后台管理两部分，为酒店提供全方位的管理和信息服务。

## 技术栈

- JDK 8
- MySQL 5.7
- IntelliJ IDEA
- Maven
- SpringBoot
- Spring
- SpringMvc
- Mybatis
- HTML
- CSS
- JavaScript
- jQuery
- Vue

## 功能模块

### 前台门户

- 登录界面
- 客房展示
- 服务介绍

### 后台管理

#### 系统角色

- 系统管理员
- 普通用户
- 会员
- 清洁人员

#### 功能介绍

- **用户管理**：创建、编辑和删除用户账户，管理权限和角色。
- **会员管理**：录入会员信息，设置会员等级和权益。
- **清洁人员管理**：分配清洁任务，监控清洁进度。
- **用户预约管理**：处理用户的预约请求。
- **会员预约管理**：提供会员专享房型和优先预订权益。
- **用户取消管理**：处理用户取消预订或预约请求。
- **会员取消管理**：处理会员专享的取消请求。
- **用户入住管理**：进行入住登记，分配房间。
- **会员入住管理**：提供会员专享的入住礼遇。
- **用户退房管理**：处理退房请求，结算费用。
- **会员退房管理**：处理会员退房流程，包括积分结算。
- **清扫房间管理**：监控客房清洁状态，分配清扫任务。
- **留言板管理**：为客人提供意见和需求留言平台。
